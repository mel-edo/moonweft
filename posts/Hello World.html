So I finally made the blog section

After being recommended a lot of videos on Youtube about indie web and neocities and indie web styled websites, I finally got around to building my own website and so far it's been pretty intuitive. HTML and CSS are pretty easy to get the hang of and i've been having a lot of fun.

Tech stack

I spent a while thinking about how to make this website. At first, I thought about using some static site generator like jekyll or zola.

I settled on just plain ass handwritten html and css with a bit of js for some dynamic elements. A big part of this was to learn some webdev (barfs violently). Be grateful it's not React lmao.

This blog section is also pretty cool. I write these in obsidian in markdown and a python script converts this into a blog post on the website which is pretty awesome.

The design

I've made each section of this website feel like a different piece of software/hardware.

  • The Projects page is inspired by Persona 5, one of my favourite games and probably my favourite part of this website.

  • The Links page is inspired by early Y2K/early web style.

  • This Notes section is supposed to look like audio equipment inspired by Bocchi The Rock.

Best viewed on a fork of Firefox, with eyes on a CRT screen.
